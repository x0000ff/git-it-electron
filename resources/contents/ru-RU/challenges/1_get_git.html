Установить Git на твой компьютер и настроить имя и электронную почту.

Git

Git это бесплатное открытое программное обеспечение (исходный код доступен публично), написанное Линусом Торвальдсом (Linus Torvalds), который также написал ядро операционной системы Linux.

Git это программа для слежения за изменениями. В программировании это называется контролем версий. Если вы уже работали в каком-то текстовом редакторе с поддержкой контроля за изменениями, то вы уже знакомы с этим понятием!

Установка Git

Мы рекомендуем установить Git на ваш компьютер посредством установки программы GitHub Desktop. Мы не будем использовать эту программу в Git-it (мы учимся работе с Git в терминале!), но GitHub Desktop содержит Git и, это самый простой путь установить Git на любую платформу.

  • Windows: Используй Git Shell в Командной строке (cmd).
  • Mac: Открой GitHub Desktop в в меню выбери пункт Install Command Line Tool... для установки инструментов командной строки. Используй Terminal или любую другую программу командной строки (например, iTerm2).

Не уверен, установлен ли Git? Набери в твоём терминале git --version и, если команда вернёт номер версии 1.7.10 или выше, то всё уже установлено! Вы можете найти больше информации на 1.7.10официальном сайте Git.

Программное обеспечение Git

Настольная программа GitHub Desktop может делать многое с Git, но не всё, именно поэтому так важно уметь работать с Git в терминале. Но научившись этому, настольная программа GitHub Desktop вам очень пригодится, потому что она организует и лучше визуализирует информацию ваших проектов, например, так как делает сайт GitHub.

Git сам по себе не похож на обычные программы. Возможно, что вы даже не увидите его иконку на Рабочем столе, но он всегда доступен в терминале или посредством программы GitHub desktop.

Настройка Git

Как толко вы установили GitHub Desktop (и Git вместе с ним), откройте вам терминал. Вы модете проверить, что Git действительно доступен выполнив в терминале:

git --version

Эта команда вернёт вам версию Git на вашем компьютере, что-то вроде:

git version 1.9.1

Затем, настроим Git, чтобы он знал что работает с Вами:

Настроим Ваше имя:

git config --global user.name "Ваше Имя"

Настроим Вашу электронную почту:

git config --global user.email "ваша@почта.com"

Вы закончили первое испытание! Нажмите кнопку 'Verify' для проверки.
